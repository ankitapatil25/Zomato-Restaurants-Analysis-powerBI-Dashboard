# 🍽️ Zomato Restaurants Analysis Dashboard

## 📌 Project Overview

This project presents an interactive Power BI dashboard developed using the Zomato Restaurants dataset. The dashboard provides comprehensive insights into restaurant distribution, cuisines, customer ratings, votes, online delivery services, and table booking availability across multiple countries and cities.

The objective of this project is to help stakeholders understand restaurant performance, customer preferences, and market trends through data-driven visualizations and key performance indicators.

---

## 🎯 Business Objective

The goal of this analysis is to:

- Analyze restaurant distribution across cities and countries.
- Identify the most popular cuisines.
- Understand customer ratings and voting behavior.
- Evaluate the impact of online delivery and table booking services.
- Discover trends that can support restaurant business decisions.

## 📈 Dashboard Visualizations

### 1. City-wise Restaurant Distribution
- Shows the number of restaurants available in each city.
- Helps identify cities with the highest restaurant concentration.

### 2. Cuisine-wise Restaurant Analysis
- Displays the most popular cuisines based on restaurant count.
- Highlights customer food preferences.

### 3. Restaurant Rating Analysis
- Compares restaurant ratings against cuisine offerings.
- Helps identify highly rated restaurants.

### 4. Online Delivery Analysis
- Analyzes the proportion of restaurants offering online delivery.
- Evaluates digital service adoption.

### 5. Table Booking Analysis
- Shows the percentage of restaurants providing table booking facilities.
- Measures customer convenience services.

### 6. Country-wise and Year-wise Filtering
- Interactive slicers allow detailed analysis across countries and years.

---

## 🔍 Key Insights

- New Delhi has the highest concentration of restaurants among all cities.
- North Indian cuisine is the most dominant cuisine category.
- A majority of restaurants do not provide table booking facilities.
- Online delivery services are available in a relatively smaller portion of restaurants.
- Customer votes and ratings can be used to identify high-performing restaurants.
